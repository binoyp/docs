## Coming Soon

The SecureLoad REST API will allow you to integrate loading computer data and workflows directly into your own systems — vessel management platforms, ERP systems, port community systems, and custom tooling.

## Planned Capabilities

<CardGroup cols={2}>
  <Card title="Loading Conditions" icon="scale-balanced">
    Read and write loading conditions, including cargo, ballast, and stability results.
  </Card>
  <Card title="Vessel Data" icon="ship">
    Access vessel hydrostatic data, tank tables, and approved loading manual parameters.
  </Card>
  <Card title="Plan Exchange" icon="arrows-left-right">
    Submit and retrieve loading plans for ship–shore data exchange workflows.
  </Card>
  <Card title="Reports" icon="file-chart-column">
    Generate and download stability reports and loading documentation programmatically.
  </Card>
</CardGroup>
