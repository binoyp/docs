The SecureLoad Loading Computer is a single calculation and workflow engine that runs identically across all deployment targets — onboard a vessel or in the cloud. Every feature described in this section applies to both the onboard and shore editions.

## Key Highlights

- **Single calculation engine** — the same hydrostatic, stability, and strength solver used everywhere
- **Step-based workflows** — guided operations for loading, ballast, and verification tasks
- **Mesh-based cargo definition** — place cargo in holds, on deck, or at custom positions with full 3D support
- **Real-time monitoring** — live sensor data integration for tank levels and draught readings
